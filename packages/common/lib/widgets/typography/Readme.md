Typography example:

```jsx
    <Typography>
        Default text size 16px weight 400
    </Typography>

    <Typography
        size={18}
        variant="extraLight"
    >
        Extra extra light text variant
    </Typography>

    <Typography
        size={18}
        variant="light"
    >
        Extra light text variant
    </Typography>

    <Typography
        size={18}
        variant="semiBold"
    >
        Semi bold text variant
    </Typography>

    <Typography
        size={18}
        variant="bold"
        secondary
    >
        Bold and secondary text variant
    </Typography>
```
